# Van Andel Institute Optical Imaging Core (OIC)

Welcome to the GitHub home of the **Optical Imaging Core (OIC)** at the **Van Andel Institute (VAI)**.

Our core serves the VAI community and collaborating institutions by providing access to state-of-the-art imaging resources 
and the expertise needed to train researchers to use these tools effectively. We support a broad variety of experimental and 
biological questions, ranging from gene expression to cell motility.

This organization serves as our central hub for sharing custom bioimage informatics workflows and analysis tools developed by our core staff.

---

### Working with the Core

We welcome collaboration and feedback from both our internal user base and the broader open-source bioimaging community.

* **Bug Reports & Feature Requests:** If you run into issues with one of our pipelines, or need a specific feature modified for your project, please open an issue directly in that repository's Issues tab.
* **Contributions:** Pull requests are highly encouraged! Simply fork the repository, make your changes, and submit a PR.

---

### Accessing Core Services

If you are a researcher looking to initiate a dedicated image analysis project or other imaging services, please contact us!

* **Internal VAI Requests:** If you are an internal VAI lab, please submit a request directly through our [Jira Service Desk Portal](https://varioic.atlassian.net/servicedesk/customer/portal/1).
